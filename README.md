# student-performance-analytics

## Overview
This project analyzes factors that affect student exam performance using data analytics tools. The goal is to understand how different variables such as study time, attendance, and motivation influence student results.

## Dataset
The dataset contains over 6,600 student records. It includes features such as:
- Hours studied
- Attendance
- Sleep hours
- Parental involvement
- Motivation level
- Access to resources
- Exam score

## Process

### 1. Data Cleaning
The data was cleaned using Python. Missing values were handled and columns were formatted correctly.

### 2. Data Storage
The cleaned dataset was uploaded to Supabase, which uses PostgreSQL.

### 3. Data Analysis
SQL queries were used to explore the data and calculate averages, distributions, and relationships.

### 4. Visualization
Metabase was used to create dashboards and charts such as:
- Scatter plot for study time vs exam score
- Bar charts for attendance and motivation
- Distribution of exam scores
- Pass rate analysis

## Key Insights
- Study time has a strong positive relationship with exam performance
- Higher attendance leads to better scores
- Most students score between 60 and 70
- Motivation and resources have some impact but less than study time
